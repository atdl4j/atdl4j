atdlj - The Open Source Java Solution for FIXatdlSM

atdl4jSM is an open-source, not-for-profit, industry-driven implementation of FIXatdlSM, the FIX ProtocolSM Algorithmic Trading Definition Language standard developed by FIX Protocol Limited (FPL). The atdl4j platform is the result of collaboration between both Buy-side and Sell-side innovators within the Electronic Trading industry, and is integrated within the trading systems of several major investment banks and FIX providers. The core goal of the atdl4j project is to promote open-source standards within the financial industry, and to increase the adoption rate of the FIXatdl standard itself. While the atdl4j open source effort encourages the use of and implements FIXatdl, it is an independent effort and not directly affiliated with the FIX Protocol Limited organization.

Key Features:
  • Displays "algo screens" based on the industry-standard FIXatdl XML schema developed by FPL
  • Can be run as a standalone testing/validation tool or dropped in as a component within one's trading system
  • Extracts and generates the algo-specific FIX message content
  • Supports pre-loading algo settings (eg Cancel/Replace) and 'input data' used for strategy filtering and rule referencing
  • Handles message validation, visibility, and enablement rules
  • Written in the Java language using standard libraries
  • Supports the full set of GUI widgets defined in FIXatdl 1.1
  • Structured to be extensible (eg Swing in addition to SWT, override/replace widgets, etc)
